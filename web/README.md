# وب اپ قرآن کریم

یک وب اپلیکیشن کاملا آفلاین برای مطالعه قرآن.

## نحوه استفاده

### روش 1: باز کردن مستقیم
فقط فایل `index.html` رو با مرورگر باز کن. همین!

### روش 2: نصب به عنوان PWA
1. فایل `index.html` رو با Chrome یا Edge باز کن
2. روی آیکون نصب (Install) کلیک کن
3. اپ مثل یه اپ واقعی روی گوشی/کامپیوتر نصب میشه

### روش 3: هاست کردن آنلاین
```bash
# با Python
python -m http.server 8000

# با Node.js
npx serve .

# یا آپلود کن به هر هاست رایگان:
# - GitHub Pages
# - Netlify
# - Vercel
```

## ویژگی‌ها

✓ کاملا آفلاین (بدون نیاز به اینترنت)
✓ قابل نصب روی موبایل و دسکتاپ
✓ سبک و سریع (کمتر از 100KB)
✓ سه تب: نشان شده، سوره‌ها، جزها
✓ نشان‌گذاری سوره‌ها
✓ رابط کاربری فارسی

## فایل‌ها

- `index.html` - صفحه اصلی
- `app.js` - منطق اپلیکیشن
- `data.js` - داده‌های قرآن
- `sw.js` - Service Worker برای آفلاین
- `manifest.json` - تنظیمات PWA

## نصب روی موبایل

1. فایل‌ها رو روی یه سرور بذار (GitHub Pages رایگانه)
2. با گوشی لینک رو باز کن
3. از منوی مرورگر "Add to Home Screen" رو بزن
4. اپ روی صفحه اصلی گوشی نصب میشه

تمام!
